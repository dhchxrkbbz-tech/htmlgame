# Vertical Slice Walkthrough

## Target Demo Length

- 3-5 minutes with one presenter and an optional second client joining in the final step.

## Primary Route

1. Login or enter as guest, then land in Adventurer Camp.
2. Follow the landmark route east until Crystal Path is visible.
3. Fight at least two enemies in the grove, then pick up one loot drop with `E`.
4. Open the trader with `M`, then either list one item or buy one item.
5. Open the social panel with `P` or `G`, then create or join a guild.
6. Send one guild chat message to show the live chat/feed update.
7. Join the same instance from a second browser window or launcher session and verify the remote player appears.

## Presenter Notes

- The HUD objective tracker mirrors this route during play.
- The second-client step is intentionally last so the single-client slice is already complete before multiplayer handoff.
- If the local server is unavailable, combat and movement still work, but social and shared market validation require the server.
